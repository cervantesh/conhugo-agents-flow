# Coordination Docs

This folder does not contain product documentation.

It contains working-memory artifacts for the development process:

- active context
- agent handoffs
- operational and technical working decisions

## Boundary

These files are:

- not application runtime
- not end-user documentation
- not customer-facing specifications
- not production configuration

## Purpose

This folder exists to keep durable shared context inside the repository being worked on and reduce drift between humans and agents during development.
